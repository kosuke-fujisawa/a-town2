## 中心テーマ

**「街は生き物である」**

住人が増えれば、価値観も増える。

- 古い住人 / 新しい移住者
- 外から来た文化
- 商業化・効率化・観光化

「居心地の良かった街」が少しずつ変質していく。しかし変化を止めることもまた衰退を意味する。本作はこのジレンマを扱う。

### サブテーマ

- 継続
- 共同体
- 時間経過・世代
- 喪失と継承
- 「日常は完成しない」

## 物語構造

**大事件は起きない。**

魔王も世界滅亡も存在しない。代わりに「小さな不可逆」を積み重ねる。

- 誰かが引っ越す
- 店が潰れる
- 子どもが進学する
- 恋人が夢を追う
- 祭りの形が変わる

## ゲームループ

### 日常ループ

```
ゲームループ外へ行く（仕事・探索）
        ↓
    家へ帰る
        ↓
  三人で過ごす
        ↓
  関係性が変化する
        ↓
    翌日へ
```

### 街スケールのループ

```
日常（仕事・交流・店運営・家庭生活）
        ↓
  街の変化イベント
  （新住人・店舗増加・トラブル・開発・祭り・移住希望）
        ↓
   プレイヤーの判断
   （誰を受け入れるか / 何を残すか / 何を変えるか）
        ↓
    街の空気が変わる
  （賑やかになる・静かになる・人が去る・新しい文化）
```

## 三人暮らしシステム

### コンセプト

> 「三人で暮らす家」

1が「親密さ」を描くなら、2は以下を描く：

- 距離感
- バランス
- 空気の揺れ
- 役割分担
- 居場所

**「共同体の最小単位」**として機能する。

### 1との決定的違い

**「二人きりになれない」**

三人暮らしは：

- 一対一ではない
- 常に空気が変化する
- 関係性が循環する

例：

- 一人が料理
- 一人が風呂
- 一人が読書

→ 空間の"ズレ"が常に発生する。

### 生活設計

以下の要素を具体的に描く：

- 家事分担
- 家具共有
- 冷蔵庫
- 金銭感覚
- 睡眠時間
- 来客

### 感情設計

ドラマチックな修羅場ではなく、「生活の摩擦」として描く。

扱う感情：

- 疎外感
- 気まずさ
- 遠慮
- 空気読み
- 役割固定化

### 恋愛構造

2では恋愛が中心ではなく、**三人の空気感が主役**。

> 「誰を好きか」より、「三人でどう暮らすか」が重要。

## キャラクター・関係性設計

### 設計思想

**「人は変わる」**

前作では比較的静的だったキャラクターを、本作では時間経過によって変化させる。

重要なのは **「イベントで完結しない関係性」**。

### 変化の例

| キャラクター | 変化 |
|---|---|
| 子ども | 成長する |
| 老人 | 衰える |
| 夫婦 | 関係が変化する |
| 移住者 | 馴染めず悩む |
| 店主 | 夢を諦める |
| 恋人 | 倦怠期を迎える |

### 恋愛システム

**「結ばれて終わり」ではない。**

恋愛成立後も関係は続く：

- 一緒に暮らす
- 価値観の違い
- 将来への不安
- 仕事との両立
- 家族との距離

"その後"を描く。

---

## アート方針

**「少し変わってしまった街」**

前作より街は賑やかになっている。しかし同時に：

- 昔の店が消えた
- 新しい看板が増えた
- 灯りが明るくなった
- 夜が静かではなくなった

**発展と喪失を同時に描く。**
